# Báo cáo tiến độ – Tuần 04
 
## Thời gian
Tuần 4
 
## Nội dung thực hiện
- Tìm hiểu về ADO.NET trong ASP.NET
- Tìm hiểu các đối tượng truy cập cơ sở dữ liệu
- Nghiên cứu mô hình Data Provider
- Làm việc với các đối tượng:
  - SqlConnection
  - SqlCommand
  - SqlDataAdapter
  - DataSet
- Thực hành kết nối ASP.NET với cơ sở dữ liệu SQL Server
 
## Kết quả đạt được
- Hiểu được quy trình làm việc với cơ sở dữ liệu trong ASP.NET thông qua ADO.NET
- Biết cách tạo kết nối đến cơ sở dữ liệu SQL Server
- Biết cách truy vấn dữ liệu và đưa dữ liệu vào DataSet
- Có thể áp dụng ADO.NET cho các chức năng của đồ án
 
## Khó khăn
- Cần thời gian để nắm vững cách sử dụng các đối tượng ADO.NET
- Việc xử lý dữ liệu còn cần thực hành thêm
 
## Kế hoạch tuần tiếp theo
- Áp dụng ADO.NET vào website
- Xây dựng chức năng hiển thị dữ liệu từ cơ sở dữ liệu
- Chuẩn bị xây dựng các chức năng thêm, sửa, xóa dữ liệu
