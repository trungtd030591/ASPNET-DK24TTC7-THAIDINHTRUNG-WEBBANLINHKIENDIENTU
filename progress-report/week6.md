# Báo cáo tiến độ – Tuần 06
## Thời gian
Tuần 6
## Nội dung thực hiện
- Tìm hiểu khái niệm Web Server trong ASP.NET
- Nghiên cứu Internet Information Services (IIS) của Microsoft
- Tìm hiểu các bước cài đặt và cấu hình IIS trên Windows
- Thực hành kiểm tra Web Server thông qua địa chỉ localhost
- Tìm hiểu thư mục mặc định của IIS (C:\inetpub\wwwroot)
- Tìm hiểu cách triển khai ứng dụng Web ASP.NET trên IIS
- Nghiên cứu tạo Virtual Directory để ánh xạ website lên IIS
## Kết quả đạt được
- Hiểu được vai trò của Web Server trong việc triển khai ứng dụng ASP.NET
- Biết cách kiểm tra trạng thái hoạt động của IIS
- Nắm được cách triển khai website ASP.NET trên IIS bằng thư mục wwwroot
- Hiểu được cách cấu hình và quản lý website thông qua IIS Manager
- Có thể chuẩn bị môi trường để triển khai đồ án trên Web Server
## Khó khăn
- Việc cấu hình IIS ban đầu còn phức tạp
- Cần thực hành thêm để làm quen với các chức năng quản lý Web Server
## Kế hoạch tuần tiếp theo
- Triển khai website đồ án lên IIS
- Kiểm thử hoạt động của website trên Web Server
- Hoàn thiện chức năng và nội dung website
